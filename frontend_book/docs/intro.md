# AI/Spec-driven Technical Book: Course Introduction

## Welcome to the AI/Spec-driven Technical Book Platform

This documentation site hosts educational content for AI and robotics students, focusing on practical implementation of advanced concepts. Our courses are designed to bridge the gap between theoretical knowledge and real-world applications.

## Course Modules

Currently, we offer the following course modules:

- **ROS 2 Robotic Nervous System**: Learn how ROS 2 functions as the middleware nervous system of humanoid robots, communication primitives for robot control, and structural modeling using URDF.

## Prerequisites

Before starting any course, you should have:

- Basic programming knowledge (Python recommended)
- Fundamental understanding of AI concepts
- Familiarity with command line tools
- Basic understanding of software development concepts

## How to Use This Platform

1. Navigate to the course module you're interested in using the sidebar
2. Follow the chapters in order for best learning experience
3. Complete the exercises at the end of each chapter
4. Apply the concepts in practical projects

## Course Structure

Each course module is organized as:
- Foundational concepts
- Practical implementation
- Hands-on exercises
- Advanced topics
