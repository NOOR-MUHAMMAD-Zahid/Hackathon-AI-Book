# Intent Processing Flow Diagram

[Diagram showing the flow from voice command to structured robot intent]

## Description
This diagram shows the process of converting natural language voice commands into structured robot intents that can be executed by the robot's action system. The flow includes transcription, entity extraction, intent classification, and action mapping.

## Key Elements
- Raw Voice Input
- Transcription (Text Conversion)
- Entity Recognition
- Intent Classification
- Action Mapping
- ROS 2 Message Formation
- Action Execution
- Validation Layer
